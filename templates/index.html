CryptoSuite

Educational app demonstrating classical ciphers: Caesar, Playfair, Hill (2×2), and One-Time Pad (OTP).

Quick start
  1. Choose a cipher from the navigation.
  2. Enter plaintext and a key (or generate one).
  3. Encrypt or Decrypt and view results.
Suggested test values
  • Caesar: Plain = ATTACK AT DAWN, Shift = 3
  • Playfair: Keyword = MONARCHY, Plain = BALLOON
  • Hill (2×2): Key = 3 3 2 5, Plain = HELP
  • OTP: Plain = HELLO (Generate key then decrypt using given key)
