# CryptoSuite
CryptoSuite is a Flask-based web application designed to demonstrate the working of classical cryptographic algorithms in an interactive way. It allows users to encrypt and decrypt messages using well-known techniques such as the Caesar Cipher, Playfair Cipher, Hill Cipher, and One-Time Pad.

The project aims to help students and beginners understand encryption principles like substitution, transposition, matrix-based encoding, and random key generation. Each algorithm is implemented with both encryption and decryption functionalities, showcasing how plaintext is transformed into ciphertext and vice versa.

The web interface provides a simple and user-friendly environment to test different keys, view outputs instantly, and learn the mathematical or logical foundation behind each algorithm.
